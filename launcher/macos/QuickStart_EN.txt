WuZi Gomoku macOS Portable

Recommended usage:
1. Extract WuZiLauncher-macos-v1.0.0.tar.gz
2. Double-click WuZiLauncher.app
3. If macOS blocks the first run, allow it in System Settings -> Privacy & Security
4. Double-click Stop-WuZi.command to stop the local server

Compatibility notes:
- You can also launch via Start-WuZi.command
- Requires Node.js or Python 3
- Runtime logs are stored in the .runtime-macos folder
